# DIABETES PREDICTION PROJECT

## Project Overview:

This is a Machine Learning based web application that predicts whether a person is diabetic or non-diabetic based on medical input parameters.

The system uses a trained classification model and provides real-time predictions through a web interface.

## Features:

* User input form
* Machine Learning prediction
* Real-time result display
* Simple and clean interface
* Input validation

## Input Parameters:

* Pregnancies
* Glucose Level
* Blood Pressure
* Skin Thickness
* Insulin
* BMI (Body Mass Index)
* Diabetes Pedigree Function
* Age

## Prediction Output:

0  -> Non-Diabetic
1  -> Diabetic

## Technologies Used:

Backend:

* Python
* django
* Scikit-learn
* Pandas
* NumPy

Frontend:

* HTML
  

## How to Run:

1. Install required libraries:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open browser and go to:
   http://127.0.0.1:5000/
